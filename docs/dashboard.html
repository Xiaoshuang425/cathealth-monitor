立即行动

  • 收集粪便样本：1-2小时内的新鲜粪便，用保鲜袋装好
  • 立即就医：带样本前往宠物医院进行粪便检查
  • 不要盲目驱虫：等待便检结果确定寄生虫种类
  • 隔离处理：如果有多只猫咪，请暂时隔离

兽医治疗流程

  • 粪便检查：显微镜鉴定寄生虫种类（蛔虫、钩虫、绦虫、球虫、贾第鞭毛虫）
  • 针对性用药：根据检查结果使用处方驱虫药
  • 完成疗程：即使症状消失也要完成整个治疗周期
  • 复查确认：治疗后复查确保寄生虫完全清除

常见寄生虫对应药物

  • 蛔虫/钩虫：吡喹酮、芬苯达唑、米尔贝肟
  • 绦虫：吡喹酮、艾沙达唑（需同时进行跳蚤防治）
  • 球虫：托曲珠利、磺胺类药物
  • 贾第鞭毛虫：芬苯达唑、甲硝唑

预防措施

  • 定期驱虫：室内猫每3个月，室外猫每月进行预防性驱虫
  • 环境清洁：每天清理猫砂盆，定期消毒家居环境
  • 跳蚤防治：坚持使用体外驱虫药，杜绝绦虫感染源
  • 饮食安全：避免喂食未经处理的生肉/生鱼
